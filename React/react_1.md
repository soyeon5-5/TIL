## 리액트 설치 및 기초

1. Node.js 설치

2. cmd에서 npm install

3. 버전 확인 : npm -v 

   ​					node -v

4. npx 1회성 최신 패키지 설치 npm install -g

5. 버전 확인 : --version

6. 리액트 폴더를 만든 후 해당 폴더 위치로 cmd에서 설정한 후

   npx create-react-app 'reatapp'  입력, ' '부분은 생성될 폴더의 이름

   혹시 에러가 난다면 다른 이름으로 입력

7. 크롬 웹 스토어에서 React-Developer Tools 설치해서 사용하면 편리
